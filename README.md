# Steve's Lab Hub

This is a Next.js project for managing college lab programs, notes, and other academic materials. It was created with Firebase Studio and is ready for deployment.

## Features

- Browse lab programs by subject, year, and semester.
- Filter and search for specific programs and materials.
- Copy code snippets with one click.
- Run web-based programs (HTML/CSS/JS) directly in the browser.
- A collection of tech-themed games to test your skills.
- Easily updatable with new content by modifying simple data files.
